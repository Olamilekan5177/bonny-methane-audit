# Nigeria LNG (Bonny Island) Methane Monitoring Audit

## Project Overview
This project uses **Satellite Spectroscopy (Sentinel-5P TROPOMI)** to monitor atmospheric methane concentrations ($CH_4$) around the Bonny Island NLNG terminal. 

## Key Features
* **Interactive Split-Map:** Compare high-resolution satellite imagery with methane heatmaps.
* **Infrastructure Correlation:** Identified methane hotspots (1915+ ppb) overlapping with industrial pipeline corridors.
* **Environmental Impact:** Analyzed 2024 median methane levels to distinguish between background signals and industrial leaks.

## Tech Stack
* **Engine:** Google Earth Engine (Python API)
* **Visualization:** Leafmap, Folium
* **Data Source:** Copernicus Sentinel-5P
